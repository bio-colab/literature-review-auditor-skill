# الحقن الجراحيّ للتعليقات (المرحلة 7)

الهدف: إعادة الملف وقد وُسِمت ملاحظاتُ التأصيل **داخل الملف نفسه** — تعليقاتٌ على مواضع
الفجوة والتلخيص واليتامى والتقادم — **دون تفكيك الملف**، حفاظاً على التنسيق. نعتمد
أدوات مهارة `docx` نفسها، ونعيد استخدام `place_marker.py` من `citation-auditor` (لا
نكرّرها).

> اسأل الباحث أولاً: تعليقاتٌ فقط (الأأمن، غير مدمّر)؟ **الافتراض هنا تعليقاتٌ لا
> تصحيحات:** ملاحظاتُ هذه المهارة حِجاجية (أعِد التأليف، وسّع النقد، خفّف الدعوى) لا
> نصّية جاهزة — والجدار الأخلاقيّ يمنع أن نكتب البديل في المتن. فالتعليقُ يدلّ، والباحثُ
> ينفّذ.

## سير العمل

```bash
DOCX=/mnt/skills/public/docx/scripts
CIT=/mnt/skills/user/citation-auditor/scripts   # نعيد استخدام place_marker.py

# 1) فكّ الملف وأزل الروابط الرمزية
unzip -q file.docx -d unpacked/ && find unpacked -type l -delete

# 2) ادمج الـ runs كي يصير النصّ المستهدَف قابلاً للإيجاد
python $DOCX/merge_runs.py unpacked/

# 3) لكل ملاحظة: أنشئ تعليقاً ثم رسّخ علامته حول الموضع (جملة الفجوة، فقرة الاستعراض…)
python $DOCX/comment.py unpacked/ "🟠 فجوة موضعية بلبوس معرفية: الموضوع مدروسٌ عالمياً؛ صُغ التميّز بوصفه سياقاً محلّياً، وأفِد من الأدبيات العالمية."   # → id=0
python $CIT/place_marker.py unpacked/word/document.xml --id 0 --target "لم تتناول أيُّ دراسة"

python $DOCX/comment.py unpacked/ "🟠 استعراضٌ تلخيصيّ: الفقرات سردٌ متجاور بلا محور. أعِد تنظيم الدراسات 2 و5 و7 تحت محورٍ يقابل نتائجها."   # → id=1
python $CIT/place_marker.py unpacked/word/document.xml --id 1 --target "تناول الباحث"

# 4) أعد البناء ثم تحقّق (إلزاميّ)
(cd unpacked && rm -f ../out.docx && zip -Xr ../out.docx .)
python $DOCX/office/validate.py out.docx --original file.docx
```

## ملاحظات حاسمة (مطابقةٌ لسير `citation-auditor`)
- **رتّب: أنشئ التعليق ثم رسّخ علامته فوراً**، ثم انتقل. الأرقام تُسنَد تصاعدياً بترتيب
  استدعاء `comment.py`.
- **`merge_runs.py` شرطٌ للترسيخ.** إن فشل `place_marker.py` في إيجاد الهدف، قصّر النصّ
  الهدف إلى شذرةٍ مميّزةٍ متصلة (مطلعُ جملة الفجوة، لا الجملة كاملة).
- **العلامات أبناءٌ مباشرون لـ `<w:p>`.** الأداة تلتزم بهذا.
- **التحقّق غير قابل للتخطّي:** أيُّ «Broken reference» يجعل الملف يبدو تالفاً في وورد.

## ماذا تكتب في التعليق
مكتفياً بذاته، قابلاً للتنفيذ، **دالّاً لا مُنجِزاً** (الجدار الأخلاقيّ):
> [شدّة] الخلل → لماذا هو خلل (بعدسة العائلة/المعيار) → ما ينبغي أن يفعله الباحثُ **هو**.

**تجنّب** كتابة نصٍّ بديلٍ جاهز في التعليق (فقرة فجوةٍ مصوغة، محورٍ مؤلَّف) — ذلك يتجاوز
الجدار. اكتب ما يدلّ الباحثَ على الفعل، لا ما ينوب عنه فيه.
